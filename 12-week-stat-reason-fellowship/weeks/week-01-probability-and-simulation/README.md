# Week 01: Probability and simulation

## Goal

Rebuild probability as a language for uncertainty, not as a bag of formulas.

## Required watching

- Harvard Stat 110, Lectures 1-2: probability, counting, story proofs.
  - https://stat110.hsites.harvard.edu/youtube

## Required reading

- Introduction to Probability, Chapters 1-2.
  - https://probabilitybook.net/
- Stat 110 strategic practice for the same topics.
  - https://stat110.hsites.harvard.edu/strategic-practice-problems

## Project exercises

Create `project/notebooks/week-01-probability-simulation.*`.

1. Simulate 3 probability problems from the week, such as birthday collisions, dice sums, biased coin runs, or conditional draws without replacement.
2. For each problem, write the exact answer before running the simulation.
3. Run at least 3 sample sizes, for example 1,000, 10,000, and 100,000 trials.
4. Add a small table comparing exact answer, simulated estimate, absolute error, and runtime.
5. Write 2-3 sentences on when simulation helped and when exact reasoning was cleaner.

Create `project/tasks/week-01-task-ideas.md`.

1. Draft 5 `StatReasonGym` task stubs.
2. For each stub, include skill tested, prompt sketch, expected answer, verifier idea, and likely wrong answer.
3. Label each task as calculation, interpretation, or test-choice.
4. Pick the best 2 candidates to carry forward.

## Deliverable

- A notebook or script with simulations.
- A one-page note with 5 task ideas and scoring notes.
- A short note naming the 2 strongest task candidates.

## Mastery check

You should be able to explain sample space, event, probability, independence, and simulation error without looking them up.
