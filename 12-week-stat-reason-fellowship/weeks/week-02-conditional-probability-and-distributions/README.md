# Week 02: Conditional probability and distributions

## Goal

Get comfortable with conditional probability, Bayes' rule, random variables, expectation, variance, and common distributions.

## Required watching

- Harvard Stat 110, Lectures 3-5.
  - https://stat110.hsites.harvard.edu/youtube
- StatQuest videos on probability distributions, as needed.
  - https://statquest.org/video-index/

## Required reading

- Introduction to Probability, Chapters 3-4.
  - https://probabilitybook.net/

## Project exercises

Create `project/notebooks/week-02-distributions.*`.

1. Implement or notebook-sketch small generators for Bernoulli, binomial, geometric, Poisson, normal, and empirical distributions.
2. For each generator, show mean and variance from samples and compare them with the expected values.
3. Include one Bayes' rule example where the base rate changes the conclusion.

Create `project/tasks/week-02-distribution-tasks.md`.

1. Write 5 distribution tasks where the answer can be checked from parameters alone.
2. Fully specify 2 tasks using `project/task-template.md`.
3. For the 2 full tasks, write reference answers, numeric tolerances, and malformed-answer examples.
4. Add notes on likely mistakes, such as base-rate neglect, confusing conditional directions, or using a normal approximation too early.

## Deliverable

- 2 verified distribution tasks.
- A short note explaining when simulation is useful and when exact calculation is better.
- A malformed-answer example for each verified task.

## Mastery check

You should be able to derive an expected value, use Bayes' rule, and explain why a distribution fits a scenario.
